Physical-Computing-Y1

My submission for the pcomp project.
Project Description

My project is an embroidered piece that emits a calming light show when connected to the conductive stars.
Technologies Used For Final Project

Arduino Leonardo and IDE (https://www.arduino.cc/en/software) (No special libraries) 5 mini leds 2.2k ohm resistors Pin to pin wires and crocodile clips Conductive Thread Embroidery Hoop Scrap Fabric, yarn and embroidery thread
